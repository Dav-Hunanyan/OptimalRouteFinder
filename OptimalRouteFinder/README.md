# OptimalRouteFinder (WPF, C#)

Simple demo WPF application that allows entering cities and roads (distances),
selecting start/end and must-visit cities, and computing an optimal route using
Held-Karp (exact DP) for small sets and fallback heuristics for larger sets.

This project targets .NET 8 and WPF. Open the .sln in Visual Studio 2022/2023 and build.

Disclaimer: This is a compact educational implementation intended for the diploma project.
